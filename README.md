# Chat
Multi-user chat application in python

# Technologies
* socket
* PyQt5
* SQLite
* RSA
* threading
* select

# Features
* Multiple clients
* Multiple chat rooms
* Authentication system
  * Login
  * Sign up
  * Forgot password
